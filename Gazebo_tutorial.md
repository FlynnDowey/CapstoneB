## 

You can use different levels of headers using the `#` symbol:

# Heading 1
## Heading 2
### Heading 3

## Emphasis

You can emphasize text using asterisks (*) or underscores (_):

*Italic text*
_Italic text_

**Bold text**
__Bold text__

## Lists

You can create ordered and unordered lists:

### Ordered List

1. First item
2. Second item
3. Third item

### Unordered List

- Item 1
- Item 2
- Item 3
